This folder contains the code used for the analysis of the two-state time traces from the TMFM experiments by means of Maximum-Likelihood Hidden Markov Modeling (MLHMM).

The code used here is an adaptation of the function package published in:
J. D. Chodera, P. Elms, F. Noé, B. Keller, C. M. Kaiser, A. Ewall-Wice, S. Marqusee, C. Bu- stamante, and N. S. Hinrichs, “Bayesian hidden Markov model analysis of single-molecule force spectroscopy: characterizing kinetics under measurement uncertainty.,” arXiv preprint arXiv:1108.1430, 2011.

I modified the functions of Chodera et al. accordingly for the analysis of TMFM data. More specifics on my implementation of the MLHMM are given in my Dissertation at the TUM:
Matthias Schickinger, “Untersuchung der Hybridisierungskinetik kurzer DNA-Doppelstraenge mit der neuartigen Einzelmolekuel-Methode Tethered Multi-Fluorophore Motion”, 2020. (in German language)
